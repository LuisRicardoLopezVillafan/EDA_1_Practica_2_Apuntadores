# Práctica 2. Apuntadores
En esta práctica, exploraremos algunas aplicaciones para entender el uso de los apuntadores.


**Objetivos**:

- Implementar la manipulación de memoria mediante el uso de apuntadores.
- Diferenciar entre el paso de parámetros por valor y por referencia.
- Aplicar aritmética de apuntadores en estructuras lineales y bidimensionales.
- Comprender la representación binaria de los datos a nivel de bits.

**Nota importante:** Al final de cada esqueleto, hay una sección de preguntas teóricas que deben de ser contestadas junto con la entrega de la práctica. 

## Ejercicio 1. El "espejo" de memoria.
El ejercicio consiste en completar la función `invertirCadena` para que invierta el orden de los caracteres de una palabra utilizando exclusivamente apuntadores. No se permite el uso de índices del arreglo de entrada.

## Ejercicio 2. El buscador del tesoro.
El ejercicio se trata de generar una matriz aleatoria de 5x5 y encontrar cuáles son las ubicaciones en memoria del primer y segundo valores más grandes de la matriz. Adicionalmente, se debe de inspeccionar cuál es la distancia entre las posiciones físicas en memoria.

## Ejercicio  3. Contador de bits (peso de Hamming)
El ejercicio consiste en convertir a binario un número entero positivo de entrada, y contar con ayuda exclusiva de apuntadores cuál es la cantidad de 1's que aparecen en dicha expresión.
